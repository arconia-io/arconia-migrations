# Support

Thanks for using Arconia Migrations! Here's how to get help.

* **Questions and usage help:** Use [GitHub Discussions](https://github.com/arconia-io/arconia-migrations/discussions). The issue tracker is not for support questions.
* **Bugs and feature requests:** [Open an issue](https://github.com/arconia-io/arconia-migrations/issues/new/choose) with the appropriate template. Search existing issues first to avoid duplicates.
* **Documentation:** See the [Arconia Migrations documentation](https://docs.arconia.io/arconia-migrations/latest/index.html).
* **Security vulnerabilities:** Do **not** open a public issue. Follow the [Security Policy](SECURITY.md).
