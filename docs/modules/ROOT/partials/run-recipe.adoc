You can apply this recipe with any OpenRewrite-compatible tool. The tabs below show three of them: the https://docs.arconia.io/arconia-cli/latest/[Arconia CLI], the https://docs.openrewrite.org/reference/gradle-plugin-configuration[OpenRewrite Gradle plugin], and the https://docs.openrewrite.org/reference/rewrite-maven-plugin[OpenRewrite Maven plugin]. See xref:index.adoc#_running_recipes[Running Recipes] for a comparison of when to pick each one.

Gradle::
+
First, create an `init.gradle` file in your project root with the following content.
+
[source,groovy,subs="attributes+"]
----
initscript {
    repositories {
        gradlePluginPortal()
    }
    dependencies {
        classpath("org.openrewrite:plugin:latest.release")
    }
}

rootProject {
    plugins.apply(org.openrewrite.gradle.RewritePlugin)
    dependencies {
        rewrite("io.arconia.migrations:{rewrite-module}:latest.release")
    }

    configurations.named("rewrite") {
        canBeConsumed = false
    }
}
----
+
Then, run the following command.
+
[source,shell,subs="attributes+"]
----
./gradlew rewriteRun \
    --init-script init.gradle \
    -DactiveRecipe={recipe-id}
----
+
You can remove the `init.gradle` file afterwards.

Maven::
+
[source,shell,subs="attributes+"]
----
./mvnw -U org.openrewrite.maven:rewrite-maven-plugin:run \
  -Drewrite.recipeArtifactCoordinates=io.arconia.migrations:{rewrite-module}:LATEST \
  -Drewrite.activeRecipes={recipe-id}
----
